秘密基地  

英國女作家伍爾芙曾說：「一切不曾發生，直到它被記述。」我喜歡拍照，也喜歡出去玩，想把這個網站打造成自己的秘密基地，累了就來這裡休息一下，回味自己做過哪些事。

忘了自我介紹，我是林鈞唯，一位即將口試的碩二生。 近期有感觸的一句話是：「Anything could happen this year!」
